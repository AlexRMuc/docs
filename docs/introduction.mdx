Rowy is an open source platform that allows you to build on Google Cloud
Platform in minutes.

With Rowy you can  
✅ Manage Firestore data in a spreadsheet-like UI  
✅ Write Cloud Functions effortlessly in the browser  
✅ Connect to your favorite third party platforms  
and do more with a fully flexible, open source platform.

<Video url={videoUrl} playing loop muted playsinline />

## Quick try

Set up Rowy on your Google Cloud Platform or Firebase project with this
one-click deploy button.

[![Run on Google Cloud](https://deploy.cloud.run/button.svg)](https://deploy.cloud.run/?git_repo=https://github.com/rowyio/rowyRun.git)

Or you can read the [full installation guide](./setup/install).
