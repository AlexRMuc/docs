Rowy supports over 30 field types. The data types match the
[types supported by Firestore](https://firebase.google.com/docs/firestore/manage-data/data-types).

### Text

|                                                | <div style={{ width: 115 }}>Name</div> | <div style={{ width: 310 }}>Description</div>                           | <div style={{ width: 210 }}>Data&nbsp;Type</div> |
| ---------------------------------------------- | -------------------------------------- | ----------------------------------------------------------------------- | ------------------------------------------------ |
| <ShortTextIcon style={{ display: "block" }} /> | **Short Text**                         | Text displayed on a single line.                                        | `string`                                         |
| <LongTextIcon style={{ display: "block" }} />  | **Long Text**                          | Text displayed on multiple lines.                                       | `string`                                         |
| <RichTextIcon style={{ display: "block" }} />  | **Rich Text**                          | HTML edited with a [rich text editor](https://www.tiny.cloud/tinymce/). | `string`                                         |
| <EmailIcon style={{ display: "block" }} />     | **Email**                              | Email address. Not validated.                                           | `string`                                         |
| <PhoneIcon style={{ display: "block" }} />     | **Phone**                              | Phone number stored as text. Not validated.                             | `string`                                         |
| <UrlIcon style={{ display: "block" }} />       | **URL**                                | Web address. Not validated.                                             | `string`                                         |

### Select

|                                                   | <div style={{ width: 115 }}>Name</div> | <div style={{ width: 310 }}>Description</div>                                                                 | <div style={{ width: 210 }}>Data&nbsp;Type</div> |
| ------------------------------------------------- | -------------------------------------- | ------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| <SingleSelectIcon style={{ display: "block" }} /> | **Single Select**                      | Single value from predefined options. Options are searchable and users can optionally input custom values.    | <code>string&nbsp;\|&nbsp;null</code>            |
| <MultiSelectIcon style={{ display: "block" }} />  | **Multi Select**                       | Multiple values from predefined options. Options are searchable and users can optionally input custom values. | `string[]`                                       |

| <RatingIcon style={{ display: "block" }} />     | **Rating**                             | Rating displayed as stars. Max stars is configurable, default: 5 stars.                                              | `number`                                                                        |
| <SliderIcon style={{ display: "block" }} />     | **Slider**                             | Numeric value edited with a Slider. Range is configurable.                                                           | `number`                                                                        |

### Connection

### Cloud Function

|                                                 | <div style={{ width: 115 }}>Name</div> | <div style={{ width: 310 }}>Description</div>                                                                                 | <div style={{ width: 210 }}>Data&nbsp;Type</div> |
| ----------------------------------------------- | -------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------ |
| <ActionIcon style={{ display: "block" }} />     | **Action**                             | Button with pre-defined action script or triggers a Cloud Function. Optionally supports Undo and Redo.                        | `any`                                            |
| <DerivativeIcon style={{ display: "block" }} /> | **Derivative**                         | Value derived from the rest of the row’s values. Displayed using any other field type. Requires Cloud Function set up.        | `any`                                            |
| <AggregateIcon style={{ display: "block" }} />  | **Aggregate**                          | Value aggregated from a specified sub-table of the row. Displayed using any other field type. Requires Cloud Function set up. | `any`                                            |
| <StatusIcon style={{ display: "block" }} />     | **Status**                             | Displays field value as custom status text. Read-only.                                                                        | `any`                                            |
